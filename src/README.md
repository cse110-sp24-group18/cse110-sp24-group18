# Journal Navigation
## How to use:
- clone this repo
- live server index.html
## Basic Features:
- Sort by latest
- Sort by earliest
- Filter by current month
- Filter by string
- Select journal from list
- Journal added each day
- Remove Journal
- Write to journal
